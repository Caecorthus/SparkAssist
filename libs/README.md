# Local Wathe Development Dependency

SparkAssist compiles against the Spark Wathe fork version
`1.5.6-spark-1.21.1`. Place the authorized development jar here as:

```text
libs/wathe-1.5.6-spark-1.21.1.jar
```

The jar is intentionally ignored and is not redistributed by SparkAssist.
Build it from the matching
[TrainMurderMystery source](https://github.com/XruiDD/TrainMurderMystery), or
obtain it through the project's authorized distribution channel.
